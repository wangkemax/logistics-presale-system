# 3. 牛奶线设计

## 3.1 资源计算逻辑

### 计算方法论

```
每条线路所需车辆数 = ⌈日运行总时长 ÷ 单车可用时长(扣除充电/检修)⌉

每条线路所需司机数 = 车辆数 × 班次系数
  - 8h单班线路(E4): 系数=1
  - 16h双班线路(E2/A白班): 系数=2
  - 24h三班线路(E1/E3/E8等): 系数=3

备用资源 = 总车辆数×15% + 总司机数×10%
```
